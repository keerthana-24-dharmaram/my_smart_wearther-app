# my_smart_wearther-app
SESD lab test-02
I used the OpenWeatherMap API to get weather data for any city.

I created an account on openweathermap.org and copied my API key.

In JavaScript, I stored it like: const key = "af0e5af514af73bb2acd651524333fa5";

I called the API using fetch() with the URL: weather?q=city&appid=key&units=metric.
